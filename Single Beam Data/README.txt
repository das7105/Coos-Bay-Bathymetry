Single Beam Sonar collected by P. Ruggiero and J. Woods (Oregon State University) Coordinate info: Horizontal - Oregon State Plane South (meters) Vertical - NAVD88 (meters). The newer 2017-2018 data are now included here in the file repository, along with a report describing their collection. For example, these files have all been processed with an estuary average speed of sound of 1483 m/s. More details in the report.
